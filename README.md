# quotech-users
backend java for quotech users

## Dependencies to run tests
* Java >= 11

## Installation

```bash
$ git clone https://github.com/mirlabraga/quotech-users
$ cd quotech-users
```

## Build & Start

```bash
$ cd quotech-users
$ ./mvnw spring-boot:runt
```
